# Kasa – Application web avec React

---

## Contexte
Kasa est un projet réalisé dans le cadre de la formation Intégrateur Web OpenClassrooms..  
L’objectif était de développer une application web en React à partir de maquettes fournies, en mettant en place une architecture basée sur des composants et un système de navigation dynamique..

---

## Objectifs pédagogiques
- Découvrir et utiliser React
- Structurer une application en composants réutilisables
- Mettre en place un routing côté client
- Gérer l’état et les props
- Assurer une navigation fluide sans rechargement de page

---

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- React
- React Router

---

## Fonctionnalités principales
- Affichage dynamique des logements
- Navigation entre les pages via le routing
- Page de détail pour chaque logement
- Gestion des erreurs et page 404
- Interface responsive et fluide

---

## Contraintes du projet
- Utilisation obligatoire de React
- Routing côté client avec React Router
- Respect des maquettes fournies
- Aucune librairie UI externe
- Compatibilité avec les navigateurs modernes

---

## Choix techniques
- Découpage de l’interface en composants clairs et réutilisables
- Utilisation de props pour la communication entre composants
- Gestion des erreurs de navigation via une page dédiée
- Organisation du projet facilitant l’évolutivité et la maintenance

---

## Installation et lancement
Aucune installation particulière n’est nécessaire..

1. Cloner le dépôt
2. Installer les dépendances avec `npm install`
3. Lancer l’application avec `npm start`

---

## Points forts
- Architecture React claire et lisible
- Composants réutilisables
- Navigation fluide sans rechargement
- Code maintenable et structuré
- Bonne compréhension des fondamentaux de React
